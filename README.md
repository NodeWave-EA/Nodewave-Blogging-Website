# Nodewave Blogging Website

This is the repository for the Nodewave Blogging Website, a platform for sharing articles and blog posts.

## Features

- Rich text editor for creating and editing blog posts
- Responsive design for mobile and desktop
- SEO optimization
- Commenting system
- Tagging and categorization of posts
- Search functionality

## Technologies Used

- Frontend: Vue.js, TypeScript, Tailwind CSS
- Backend: Strapi CMS, Node.js, TypeScript
- Database: PostgreSQL
- Deployment: Docker, Vercel
